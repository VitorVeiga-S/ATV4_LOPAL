# Lista de Exercícios - Lógica de Programação (Operadores Lógicos)

Este repositório contém as resoluções da terceira lista de atividades de LOPAL, feito por Vitor Andrade, e Enzo Ferrari.

# **Conteúdo da lista:**

* **Aprovação Acadêmica:** Checagem de nota e frequência.
  
* **Verificação de gratuidade de entrada:** Verificação de gratuidade por idade (menor que 5 ou maior/igual a 65 anos).
  
* **Gestão de Estoque:** Verificação de disponibilidade de produto para liberação de venda.
  
* **Recrutamento:** Filtro de currículos baseado em idade mínima e competências técnicas (JavaScript ou Python)
  
* **Análise de empréstimo:** Lógica para aprovação de empréstimos considerando salário, fiador e restrições no Serasa
  
* **Sistema de Descontos:** Aplicação de "Super Desconto" através de cenários combinados de fidelidade, valor de compra e adimplência

##  Metodologia
Fizemos um fluxograma e um pseudocódigo para cada exercício, fazendo o fluxograma passo a passo e o pseudocódigo seguindo o fluxograma, para resolver os problemas propostos da forma mais intuitiva possível, usando condicionais e operadores lógicos, como "e", "ou", e "não".
